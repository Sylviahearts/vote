HOW TO VOTE

Voting is a method for a group, such as a meeting or an electorate, in order to make a collective decision or express an opinion usually following discussions, debates or election compaigns.

The following steps be followed to be able to cast your vote

1. Users need to register to be able to vote

2. If a user has already registered, the next step is to sign up

3. Each user can only vote once

4. After the election is complete, users can view the results of the election from the result section
